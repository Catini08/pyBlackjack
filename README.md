# Interactive Deck of Cards

Terminal-based Anglo-American standard deck in Python and Blackjack with a full rendering
pipeline for ASCII card glyphs.

## Purpose
Grasping Python fundamentals, built-in libraries and git.

## What it demonstrates
- Finite state machine architecture for terminal UI
- OOP design with separated rendering and game logic
- Dynamic deck management and full Blackjack rule implementation
- Accurate card picking dynamic using pop(0) and self.order to organize the deck

## Stack
Python 3 · colorama · stdlib

## Run
pip install colorama
python3 deck.py
